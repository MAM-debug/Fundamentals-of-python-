## ğŸ“š Library 1: Streamlit

**Streamlit** is a Python library for creating interactive web applications and dashboards without HTML, CSS, or JavaScript.

### What You'll Learn:
- Create interactive web apps
- Build dashboards
- Display data and visualizations
- Handle user inputs
- Deploy applications

### Installation:
```bash
pip install streamlit
```

### Run an App:
```bash
streamlit run filename.py
```
